# Images Folder

This folder contains all images used in your portfolio.

## Folder Structure

- **profile/** - Your profile photo
  - Add your profile photo here (e.g., `profile-photo.jpg`)
  - Recommended size: 400x400px to 800x800px
  - Format: JPG or PNG

- **projects/** - Project screenshots
  - Add screenshots of your projects here
  - Example files:
    - `microservices.jpg`
    - `timetable.jpg`
    - `chatbot.jpg`
  - Recommended size: 1200x800px (16:9 ratio)
  - Format: JPG or PNG

- **companies/** - Company logos (optional)
  - Add company logos if you want to display them in Work Experience
  - Example files:
    - `amdocs-logo.png`
    - `infosys-logo.png`
  - Recommended size: 200x200px or smaller
  - Format: PNG (with transparency if needed)

## Quick Start

1. **Add Profile Photo:**
   - Place your photo in `profile/profile-photo.jpg`
   - The Home section will automatically use it

2. **Add Project Screenshots:**
   - Place project screenshots in `projects/` folder
   - Update the filenames in `src/components/Projects.js` to match your files

3. **Optimize Images:**
   - Use tools like TinyPNG (https://tinypng.com/) to compress images
   - This improves page load speed

## Image Naming Tips

- Use lowercase letters
- Use hyphens instead of spaces: `profile-photo.jpg` ✅ not `profile photo.jpg` ❌
- Be descriptive: `microservices-architecture.jpg` ✅ not `img1.jpg` ❌
