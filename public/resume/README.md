# Resume Files

Place your resume files in this folder:

## Required Files

1. **Mansi_Misal_Resume.pdf** - Your resume in PDF format
2. **Mansi_Misal_Resume.csv** - Your resume data in CSV format

## File Naming

Make sure the files are named exactly:
- `Mansi_Misal_Resume.pdf`
- `Mansi_Misal_Resume.csv`

Or update the file names in `src/components/Contact.js` to match your files.

## CSV Format Example

The CSV file should contain your resume data. Example structure:

```csv
Category,Details
Name,Mansi Misal
Email,mansimisal88@gmail.com
Phone,+91XXXXXXXXXX
Location,India
Summary,Software Engineer with 4.5 years of experience...
Experience,Amdocs - Software Developer (Aug 2022 - Present)
Education,Bachelor of Engineering - Computer Science (2016-2020)
Skills,Java, Spring Boot, Microservices, Python
```

## How to Create CSV from PDF

1. Extract text from your PDF
2. Organize it into categories
3. Save as CSV format
4. Or use online PDF to CSV converters

## Notes

- PDF files are typically preferred by recruiters
- CSV format is useful for data import/parsing
- Both files will be downloadable from the Contact section
